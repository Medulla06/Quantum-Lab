# Quantum Computing Lab – Qiskit

This repository contains implementations and experiments from the **Quantum Computing Lab** using **Qiskit**. 
The lab explores fundamental quantum computing concepts such as quantum gates, entanglement, quantum algorithms, and error correction.

The objective of this lab is to gain hands-on experience with quantum circuits and algorithms using the **Qiskit framework**.

---

## Technologies Used

* Python 3.x
* Qiskit
* Jupyter Notebook / Python Scripts

---

## Content

1. Quantum Gates and Quantum Circuits
2. Quantum Communication
  * Quantum Entanglement
  * Quantum Key Distribution (QKD)
  * Quantum Teleportation

3. Quantum Algorithms I
  * Deutsch–Jozsa Algorithm
  * Grover’s Search Algorithm
  * Quantum Fourier Transform (QFT)
  * Quantum Phase Estimation (QPE)

4. HHL Algorithm (Harrow–Hassidim–Lloyd Algorithm)

5. Quantum Error Correction

6.  Variational Quantum Eigensolver (VQE)

7. Error Mitigation Techniques in Quantum Computing
